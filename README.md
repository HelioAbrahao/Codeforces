# Codeforces – Soluções

Este repositório contém minhas soluções para problemas do Codeforces, organizadas de forma simples e objetiva, com foco em aprendizado e prática para maratonas de programação.

## Objetivo

- Treinar lógica de programação e resolução de problemas.
- Aprimorar conhecimentos em C++.
- Manter um histórico de evolução.

## Organização

- As questões resolvidas individualmente estão nomeadas diretamente com o título do problema.
- Quando participo de uma maratona ou contest específico, crio uma subpasta com o nome da competição, e os exercícios ficam organizados dentro dessa pasta.

## Linguagens utilizadas

- C++ (principal)
- Python (eventualmente)
